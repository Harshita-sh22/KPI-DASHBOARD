# KPI-DASHBOARD
Lending Club Loan KPI Dashboard | Power BI

Interactive dashboard analyzing Lending Club's accepted loans from 2007 to 2018 Q4 to track key performance metrics, risk patterns, and borrower trends.
# Key Metrics & KPIs Tracked
- **Total Loans**: 10K+
- **Approval Rate**: 50.00%
- **Average DTI (Debt-to-Income)**: 19.51
- **Overall Default Rate**: 17.7%
# Key Business Insights
1. **Risk by Grade**: Default rate increases from 5% in Grade A to 30%+ in Grade G
2. **Purpose Analysis**: Debt Consolidation dominates at 60.28% of total loan volume
